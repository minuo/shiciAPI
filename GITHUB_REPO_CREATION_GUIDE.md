# GitHub仓库创建指南

## 步骤1：登录GitHub账号

1. 打开浏览器，访问 [GitHub官网](https://github.com)
2. 使用您的GitHub账号和密码登录

## 步骤2：创建新仓库

1. 登录成功后，点击右上角的「+」号图标
2. 从下拉菜单中选择「New repository」

![创建仓库入口](https://docs.github.com/assets/cb-79227/images/help/repository/repo-create.png)

## 步骤3：设置仓库信息

在创建仓库页面，填写以下信息：

1. **Repository name**：输入仓库名称，建议使用 `shiciAPI`
2. **Description**：（可选）输入仓库描述，例如「诗词API后端服务」
3. **Visibility**：选择仓库可见性
   - `Public`：公开仓库，所有人可见
   - `Private`：私有仓库，只有您和您邀请的人可见
4. **Initialize this repository with**：请不要勾选任何选项，因为我们已经在本地初始化了Git仓库
5. 点击底部的「Create repository」按钮

![仓库设置页面](https://docs.github.com/assets/cb-12757/images/help/repository/create-repository-page.png)

## 步骤4：获取仓库URL

创建成功后，您将看到新仓库的页面。在页面顶部，您可以找到仓库的URL：

1. 点击「Code」按钮
2. 确保选择「HTTPS」选项
3. 复制显示的URL，例如 `https://github.com/your-username/shiciAPI.git`

![复制仓库URL](https://docs.github.com/assets/cb-34471/images/help/repository/https-url-clone.png)

## 后续步骤

创建完仓库后，请继续按照部署指南中的说明配置本地仓库并推送代码。
